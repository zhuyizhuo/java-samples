### 12 -> 110

$$
0 * 3^0 + 1 * 3^1 + 1 * 3^2
$$
true

### 91 -> 10101

$$
1 * 3^0 + 0 * 3^1 + 1 * 3^2 + 0 * 3^3+ 1 * 3^4
$$

true

### 21 -> 210

$$
0 * 3^0 + 1 * 3^1 + 2 * 3^2
$$
