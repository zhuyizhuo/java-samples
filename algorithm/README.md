## 散列算法
散列算法一般用于生成数据的摘要信息，是一种不可逆的算法，一般适合存储密码之类的数据,
常见的散列算法如MD5、SHA等。一般进行散列时最好提供一个salt（盐）
MD5  SHA1  SHA256  SHA512
